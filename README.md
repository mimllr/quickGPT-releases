# quickGPT

A native macOS menu bar app for quick ChatGPT conversations — click the menu bar icon, or hit a
global hotkey, and a chat popover opens right where you are. No browser tab, no app switching.

## How it works

- Lives quietly in the menu bar until you need it.
- A global hotkey opens the chat popover and focuses the input, so you can start typing
  immediately; a second hotkey opens a fresh conversation.
- Optionally auto-pastes your clipboard into the input when opened, if you turn that on in
  Settings.
- Signs in with your existing ChatGPT subscription via the [Codex CLI](https://github.com/openai/codex),
  or with an OpenAI API key if you'd rather use the platform API.
- Keeps a local history of past conversations you can jump back into.
